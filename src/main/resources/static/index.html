一、关于jqgrid取行数据

取行数据有两种方法：

1、getRowData()：获取表格行渲染后的数据。

2、getLocalRow()：获取表格行原始数据。

使用场合：

1、若是前台分页，两种方法都可以用。但是使用getLocalRow()方法时，要求数据中有不重复的id字段，若没有id字段，可将某一主键字段列中添加配置"key": true,则此字段可作为id使用。

2、若后台分页，只可以用getRowData()方法。若想使用未被渲染的数据，可使用隐藏列。

添加一个列，设置属性为隐藏，format返回行数据字符串：{"label": "", "name": "relationStoreInState", "sortable": false, "hidedlg": true, "hidden": true, "formatter":function(cellValue, options, data){ return JSON.stringify(data); }

取值的方法：

先取得渲染后的行数据：var rowData = $(this).getRowData(rowid);

那么真实行数据就是：realData = JSON.parse(rowData.relationStoreInState);
