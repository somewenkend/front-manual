注 意!!

一、formatter

可通过formatter方法格式化出来想要的数据，如本页面表格中的按钮。formatter方法的参数，

cellValue：行数据中name的值；

options：jqgrid的配置项；

data：此行的原数据；

二、关于展示列合计值

1)配置表格脚部显示：footerrow: true

2)在需要的列配置中开启求合计值："totalSum": true；若需保留小数则添加属性"fixed": 2，即保留两位小数，求平均值同理

3)在表格初始化完成回调方法中刷新列合计值

4)表格中的值可以改变的情况下，不可使用全局模糊匹配，需用表格列筛选代替

三、关于添加数据行颜色

1)在需要显示的每一列都配置："cellattr": cellAddStyle

2)声明方法：cellAddStyle，返回所需的样式

注 意!!

一、关于列筛选功能

1)首先开启表格的工具栏搜索

2)需要在表头数组中进行列配置："stype": "select"

3)关闭其他列的工具栏搜索："search": false

二、若要开启含input框的全局模糊匹配，需同时开启一般全局模糊匹配

三、关于展示列平均值

1)配置表格脚部显示：footerrow: true

2)配置求合计或求平均的文本：calculationType: '平均'

3)在需要的列配置中开启求平均值："totalAve": true

4)在表格初始化完成回调方法中刷新列平均值

四、关于jqgrid单元格编辑

设置可编辑单元格的列，禁止使用formatter！！！
